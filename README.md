# Generative-AI-for-RAG-and-code-reviews

🚀 **Generative AI for RAG and Code Reviews** is a Python-based project that leverages cutting-edge Generative AI models to power:

1. **Code Reviews** using GPT models to analyze and provide feedback on code quality, best practices, and improvements.
2. **Multi-Modal AI Chatbot - My Data Assistant** 📄🎙️: An AI-powered chatbot that enables users to interact with their documents in various formats:
   - **Text-Based Formats**: PDF (`.pdf`), Text (`.txt`), PowerPoint (`.pptx`), Word (`.docx`)
   - **Audio Format**: WAV (`.wav`)

## ✨ Features
- **Automated Code Review**: Get AI-generated insights on your code for improvements and best practices.
- **Conversational AI for Documents**: Upload documents and interact with them using natural language queries.
- **Multi-Modal Processing**: Handles both text and speech inputs for a seamless user experience.
- **Fast and Scalable**: Designed for efficient and production-ready deployments.

## 🔥 Getting Started
Follow these steps to set up and run the project on your local machine:

### 1️⃣ Clone the Repository
```bash
 git clone https://github.com/NaseebGrewal/Generative-AI-for-RAG-and-code-reviews.git
 cd Generative-AI-for-RAG-and-code-reviews
```

### 2️⃣ Install Dependencies
Ensure you have Python installed, then install the required packages:
```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Application
Start the Flask server by executing:
```bash
python utils/Flask.py
```

## 🛠️ Tech Stack
- **Python** 🐍
- **Flask** 🌐
- **OpenAI GPT Models** 🤖
- **NLTK / LangChain** for NLP Processing
- **SpeechRecognition** for audio processing
- **PyMuPDF, python-docx, pptx** for document parsing

## 📌 Usage
- **Code Reviews**: Submit code snippets, and receive AI-powered feedback.
- **Document Q&A**: Upload supported document types and ask questions about their content.
- **Speech-to-Text Interaction**: Upload `.wav` files to transcribe and analyze spoken content.
